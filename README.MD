# **SWITCHBOARD**
> High-performance Easy-to-use Ergonomics Next-generation Post-modern Cross-platform AI-powered Super (And any irresponsible adjective) Gateway.

But seriously, it's a gateway aims to be really easy to use with a not bad performance. It's not designed for the most extreme use case, but
it should be good enough for most use cases. And with a human-friendly configuration user interface, a normal user with basic computer knowledge should be able to set it up in any mainstream operating system in minutes. And the most important, it's fully customizable.

## Quick Start
```bash
sb start --config ./config.toml --controller ./controller.toml
```
